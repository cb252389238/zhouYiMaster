## 产品概述

改进六爻起卦功能的三个核心体验：铜钱投掷特效、爻象实时展示、摇完卦后的结果展示流程。

## 核心功能

### 1. 铜钱投掷特效改进

- 将简单的2D旋转动画改为真实的3D翻转效果
- 铜钱使用 `rotateY` 和 `rotateX` 模拟立体翻转
- 添加透视效果、阴影和缩放增强立体感
- 每次投掷随机改变旋转圈数和速度

### 2. 爻象实时展示

- 在铜钱上方添加爻象展示区域
- 每投掷一次立即展示对应的爻（从下往上构建）
- 阳爻使用 `⚊`，阴爻使用 `⚋`（Unicode 符号）
- 老阴（6）和老阳（9）显示为红色
- 少阳（7）和少阴（8）显示为黑色
- 去掉铜钱下方的卦象描述文字

### 3. 摇完卦后流程调整

- 第六爻摇完后停留在摇卦页面，不跳转
- 在铜钱下方直接展示卦象详情
- 详情包括：本卦名称符号、卦辞、爻辞列表、变卦信息
- 提供"重新起卦"按钮

## 技术栈

- 前端：原生 HTML + CSS + JavaScript
- 动画：CSS3 3D Transform + Animation
- 已有框架：无（原生实现）

## 实现方案

### 1. CSS 动画改进

将当前简单的 `rotate` 动画改为3D翻转效果：

```css
/* 当前动画（需要替换） */
@keyframes roll {
    0% { transform: rotate(0deg) scale(1); }
    25% { transform: rotate(90deg) scale(1.05); }
    50% { transform: rotate(180deg) scale(1); }
    75% { transform: rotate(270deg) scale(1.05); }
    100% { transform: rotate(360deg) scale(1); }
}

/* 改进后的动画 */
.ly-coin-wrapper {
    perspective: 1000px; /* 3D透视 */
}

.ly-coin {
    transform-style: preserve-3d;
    backface-visibility: hidden;
}

@keyframes flip3d {
    0% { transform: rotateY(0deg) rotateX(0deg) scale(1); }
    25% { transform: rotateY(450deg) rotateX(90deg) scale(1.1); }
    50% { transform: rotateY(900deg) rotateX(180deg) scale(1); }
    75% { transform: rotateY(1350deg) rotateX(270deg) scale(1.1); }
    100% { transform: rotateY(1800deg) rotateX(360deg) scale(1); }
}
```

```javascript
// 在 tossCoins() 函数中添加爻象渲染
function renderYaoSymbol(total) {
    const isYang = (total === 7 || total === 9); // 少阳或老阳
    const isOld = (total === 6 || total === 9);  // 老阴或老阳
    
    const symbol = isYang ? '⚊' : '⚋';
    const className = isOld ? 'yao-line old' : 'yao-line normal';
    
    // 添加到爻象展示区域
    const container = document.getElementById('lyYaoDisplay');
    const yaoDiv = document.createElement('div');
    yaoDiv.className = className;
    yaoDiv.textContent = symbol;
    container.appendChild(yaoDiv);
}
```

function showLiuYaoResult() {

                <div class="yao-ci-list">...爻辞列表...</div>

            </div>
        `;
    }
    
    // 禁用投掷按钮
    document.getElementById('lyTossBtn').disabled = true;
}
