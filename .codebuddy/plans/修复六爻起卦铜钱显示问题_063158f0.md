## 用户需求

1. **铜钱变形问题**：当前铜钱显示为椭圆形，需要调整为正圆形
2. **界面撑满问题**：铜钱显示了透明背景，需要填满显示区域
3. **移除按钮**：移除页面上的"开始起卦"按钮

## 问题分析

- **铜钱变形原因**：CSS 使用 `background-size: 200% 100%`，当图片不是标准的左右二等分拼接时会导致拉伸变形
- **透明背景原因**：图片本身可能包含透明边距，background-image 无法自动裁剪
- **按钮仍在**：之前修改未成功保存或被撤销

## 解决方案

将铜钱从 `background-image` 方式改为 `<img>` 标签方式，使用 `object-fit: cover` 确保图片填满容器且不变形，通过 `object-position` 控制显示正面或反面。

## 技术方案

### 1. 移除开始起卦按钮

- 删除 `index.html` 第528-531行的 `lyStartDiv` 容器
- 修改 `app.js` 的 `initLiuYao()` 函数，移除对 `lyStartDiv` 的引用
- 删除不再需要的 `startLiuYao()` 函数

### 2. 修改铜钱HTML结构

将 `<div class="ly-coin">` 改为 `<img class="ly-coin">`：

- 三个铜钱元素改为 img 标签
- 初始 src 指向 tongqian.jpg

### 3. 修改铜钱CSS样式

使用 `object-fit` 属性替代 `background-image`：

```css
.ly-coin {
    width: 120px;
    height: 120px;
    border-radius: 50%;
    object-fit: cover;
    object-position: 0 0;  /* 正面 */
    border: 5px solid #daa520;
    box-shadow: 0 6px 12px rgba(0,0,0,0.4);
}
.ly-coin.yang {
    object-position: 100% 0;  /* 反面 */
}
.ly-coin.yin {
    object-position: 0 0;  /* 正面 */
}
```

### 4. 修改JS逻辑

- 在 `initLiuYao()` 中设置铜钱的 src 属性
- 在 `tossCoins()` 中通过 `object-position` 切换正反面
